    <h1 align="center">Clean Keystrokes</h1>
</p>

A clean and customizable keystroke display HUD for Minecraft. Shows your WASD keys, spacebar, sneak, sprint, and mouse buttons with a live mouse movement tracker.

<img src="https://github.com/user-attachments/assets/83b10e86-d9ea-4442-8672-06df0f87fa3e" alt="Clean-Keystrokes-Banner">

<h1>Features</h1>

- Displays W, A, S, D, Space, Sneak, Sprint, LMB and RMB
- Ability to disable Sneak/Sprint row
- Toggleable animation
- Option to enable tick synced key presses
- Mouse movement tracker
- Rainbow text mode
- Fully customizable colors, including presets
- Configurable via Mod Menu
